# Bellabeat Case Study
## Google Data Analytics Capstone Project

## Overview
Analysis of FitBit fitness tracker data to uncover smart device usage trends 
and provide marketing recommendations for Bellabeat, a wellness tech company for women.

## Tools Used
- Google Sheets — initial data exploration and visualization
- SQL (SQLite) — data cleaning, deduplication, and aggregation
- Python (Jupyter Notebook) — full analysis and visualizations

## Key Findings
1. Users are sedentary for ~80% of their day (951 minutes on average)
2. Average daily steps are 8,057 — below the recommended 10,000
3. More sedentary time correlates with less sleep
4. Only 24 out of 35 users tracked their sleep

## Recommendations
1. Launch inactivity reminders in the Bellabeat app and Leaf tracker
2. Show users personalized sleep vs activity correlations in the app
3. Build a marketing campaign around closing the 2,000 step gap
